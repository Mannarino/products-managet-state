Aplicación de Gestión de Productos

Esta aplicación permite a los usuarios navegar por un catálogo de productos, seleccionar ítems, agregar cantidades al carrito y revisar su orden en un resumen. Es un proyecto de práctica desarrollado en Angular con un enfoque en la arquitectura de componentes reutilizables y gestión del estado de los productos.

Tecnologías Utilizadas

  • Frontend: Angular, javascript, css, html

Funcionalidades Clave

  • Navegación entre páginas de productos.
  • Agregar productos al carrito y modificar cantidades.
  • Modal de detalles para ver más información de cada producto.
  • Barra de búsqueda para encontrar productos rápidamente.
  • Gestión del estado del carrito y productos mediante servicios globales.

Proceso de Desarrollo

La arquitectura de la aplicación está basada en componentes reutilizables, como item-card, que se utiliza en todas las páginas para mostrar productos. Además, se implementaron servicios para manejar el estado de los productos, el carrito y las notificaciones de alertas globales.

Opté por no manejar inmutabilidad para simplificar la sincronización entre el estado de los productos y el carrito, permitiendo reflejar la cantidad de productos seleccionada en todas las páginas.
